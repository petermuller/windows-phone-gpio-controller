windows-phone-gpio-controller
=============================

This is a Windows Phone app to control the Raspberry Pi GPIO.

Requirements
------------
**Phone Application**
* Windows Phone 8+

**Raspberry Pi Server**
* Python 2.7.x
* Raspberry Pi GPIO v0.5.3a+
